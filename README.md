# Illuminati

An iOS client for anynoumous mobile Secret-like social network. **WIP**

![](Screen-Login.png)
![](Screen-Feed.png)

## Installation

1. Install the pods.

  ```bash
  $ pod install
  ```

2. Rename ReactiveCocoa's `and`, `or`, and `not` methods to `AND`, `OR`, and
`NOT`, respectively.

3. Enjoy.
